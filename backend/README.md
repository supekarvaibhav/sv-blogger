# SV Blogger Backend

FastAPI backend with async SQLAlchemy, JWT authentication, and Alembic migrations.

## Setup

1. Create and activate a virtual environment.
2. Install dependencies:

```
pip install -r requirements.txt
```

3. Configure environment variables:

- Copy `.env.example` to `.env` and update values.

4. Run database migrations:

```
alembic revision --autogenerate -m "init"
alembic upgrade head
```

5. Start the server:

```
uvicorn app.main:app --reload
```

## API Docs

- Swagger UI: `http://localhost:8000/docs`
- ReDoc: `http://localhost:8000/redoc`

## Core Endpoints

Authentication
- POST `/api/auth/register`
- POST `/api/auth/login`
- POST `/api/auth/refresh`
- GET `/api/auth/me`

Blogs
- GET `/api/blogs`
- GET `/api/blogs/{id}`
- POST `/api/blogs`
- PUT `/api/blogs/{id}`
- DELETE `/api/blogs/{id}`

Comments
- POST `/api/comments`
- GET `/api/comments/{blog_id}`
- DELETE `/api/comments/{id}`

Bookmarks
- POST `/api/bookmarks`
- GET `/api/bookmarks`

Likes
- POST `/api/likes`

## Notes

- Email verification and password reset tokens are generated by the API in development mode. Plug in an email provider for production.
- File uploads are stored in `uploads/` by default.

## Deploy (Render/Railway)

1. Set environment variables from `.env.example`.
2. Run migrations: `alembic upgrade head`.
3. Start command: `uvicorn app.main:app --host 0.0.0.0 --port 8000`.
